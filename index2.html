Що таке GNU/Linux

GNU/Linux — це вільна операційна система з відкритим вихідним кодом, розроблена для комп'ютерів та серверів. Вона базується на ядрі Linux, створеному Лінусом Торвальдсом, і використовується у всьому світі через свою стабільність, безпеку та гнучкість.

Найбільш популярні дистрибутиви Linux

  • Debian
  • Fedora
  • Arch Linux
  • Linux Mint

Команди Linux CLI

  • pwd - Показує поточний каталог.
  • echo - Виводить текст або змінну в консоль.
  • ls - Показує список файлів у каталозі.
  • cd - Змінює поточний каталог.
  • touch - Створює порожній файл.
  • mkdir - Створює новий каталог.
  • cp - Копіює файли або каталоги.
  • mv - Переміщує або перейменовує файли.
  • rm - Видаляє файли або каталоги.
  • history - Показує історію виконаних команд.

BASH-скрипт для авторизації

#!/bin/bash
echo "Введіть ім'я користувача:"
read username
echo "Введіть пароль:"
read -s password

if [[ $username == "admin" && $password == "12345" ]]; then
    echo "Авторизація пройшла успішно!"
else
    echo "Невірні дані для входу!"
fi
